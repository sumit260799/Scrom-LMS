== Introduction ==  You have downloaded a package of elearning courses in the form of zip files. These zips use either the SCORM 1.2 or AICC elearning file standard, depending on your account settings. A standards-compliant learning platform, such as a Learning Management System (LMS) or Learning Experience Platform (LXP), is required to use them.      == Installation ==  Every course within this package is a separate zip file and must remain individually zipped to work. Upload each zip using the normal course-creation process in your learning platform. Some platforms also support bulk uploads. Please refer to your platform's documentation or reach out to your platform contacts for best practices as needed.      == FAQ ==  Why are these files so small?  These zips use OpenSesame's patented delivery technology and contain a unique launch URL. Upon launch, the URL redirects to the course's content, which is served on demand via a secure global CDN.  How can I tell whether these files are SCORM 1.2 or AICC?  Checking a course zip's file format is the only time it should be completely unzipped. A SCORM 1.2 course will contain three files: imsmanifest.xml, launchpage.html, and name.html. An AICC course will contain four: manifest.au, manifest.crs, manifest.cst, and manifest.des.     == Support ==  Contact OpenSesame at www.opensesame.com, by email at support@opensesame.com, or by phone at (503) 808-1268, ext. 2 (U.S.) or +44 203 744 5541 (Europe).
